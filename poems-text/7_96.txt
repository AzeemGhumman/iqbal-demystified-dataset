دل بیباک را ضرغام رنگ است
*
دل بیباک را ضرغام رنگ است
دل ترسندہ را آہو پلنگ است
اگر بیمی نداری بحر صحراست
اگر ترسی بہر موجش نہنگ است
 

*
(96)
*

*
Lions to dauntless hearts appear as sheep,
While quaking hearts see tigers in mere deer.
The sea is a plain, if you do not fear.
But if you fear, each wave contains a shark.
